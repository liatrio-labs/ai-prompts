# Check Documentation

Determine which tools are required to complete the current task. Before starting the task, look up the documentation for the relevant tools using the Context7 and Tavily MCPs. Also search the web for any additional information that may be relevant to the task and the tools.

Use the information from the documentation to adapt your approach to the task to ensure your solution conforms to the latest best practices for the tools.
